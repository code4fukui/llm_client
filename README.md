# llm_client

## usage (console)

1. run [llm_server](https://github.com/code4fukui/llm_server)
2. deno run -A example.js

## usage (web)

1. run [llm_server](https://github.com/code4fukui/llm_server)
2. open index.html with the web server

## settings

- make endpoint.js (sample)
```js
export const endpoint = "http://[::1]:5050/";
```
